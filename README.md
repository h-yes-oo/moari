# 🌱 모아리: 모여라, 동아리! 

## 🌎 시작하기
`yarn start`

## 🧩 프로젝트 개요
각 대학의 동아리를 한 눈에 모아보고, 조건에 맞게 검색할 수 있습니다. <br />
동아리 담당자를 대신하여 홍보와 리크루팅까지 해주는 서비스입니다.
